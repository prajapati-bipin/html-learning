Semantic vs Non-Semantic HTML Elements

Semantic Elements

<header>

Contains introductory content or navigation links.

<nav>

  • Home
  • About

<main>

The main content of the document.

<article>

Self-contained composition in a document, e.g. a blog post.

<section>

A thematic grouping of content, with its own heading.

<aside>

Sidebar content, often tangentially related.

<footer>

Footer for author info, links, copyright, etc.

<address>

Contact information for author/owner.

<time>

May 2, 2025

<mark>

Highlighted text
<details> + <summary>

Hidden content revealed on click.

Non-Semantic Elements

<div>

Used purely for grouping and styling.

<span>

Inline container with no meaning.

Presentational Tags

<b> — bold text (no added importance)

<i> — italic text (no added emphasis)

<u> — underlined text

<s> — strikethrough text

<font> — deprecated font styling

<center>Centered content (deprecated)
